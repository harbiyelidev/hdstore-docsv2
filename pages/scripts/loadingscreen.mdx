<Steps>
    ### Locales

    First, you need to set the language of the loading screen.

    Defualtly set to `en (English)` the actively available languages are `en (Turkish)`.

    ```js filename="hdstore-loadingscreen/config/config.js" copy showLineNumbers
    locales: 'en',
    ```

    ### Background Video

    You can change the video in the background by typing the code in the region where `change_video_id` is specified with the `video_id` in a youtube video link `https://www.youtube.com/watch?v=video_id`!

    ```js filename="hdstore-loadingscreen/config/config.js" copy showLineNumbers
    youtubeVideoId: 'change_video_id',
    ```

    ### Social Media

    You can add your social media accounts to the loading screen by typing the code below.

    ```js filename="hdstore-loadingscreen/config/config.js" copy showLineNumbers
    ---@param name: Name of the social media
    ---@param image: Name of the image in the `public/assets/images` folder
    ---@param description: Description of the social media
    ---@param link: Link of the social media
    socials: [
        { name: 'X', image: 'twitter', description: 'x.com/hdstore', link: 'https://x.com/hdstore' },
        { name: 'Instagram', image: 'instagram', description: 'instagram.com/hdstore', link: 'https://instagram.com/hdstore' },
        { name: 'Github', image: 'github', description: 'github.com/hdstore', link: 'https://github.com/hdstore' },
    ],
    ```

    ### Staff

    You can add your staff to the loading screen by typing the code below.

    ```js filename="hdstore-loadingscreen/config/config.js" copy showLineNumbers

    ---@param image: Name of the image in the `config/team` folder
    ---@param firstname: First name of the staff
    ---@param lastname: Last name of the staff
    ---@param tag: Tag of the staff
    ---@param color: Color of the tag
    staffs: [
        { image: 'harbiyeli.png', firstname: 'Harbiyeli', lastname: 'Developer', tag: 'Owner', color: '#FFBB00' },
        { image: 'vezironi.png', firstname: 'Vezir', lastname: 'Oni', tag: 'Co-Owner', color: '#FF0000' },
        { image: 'harbiyeli.png', firstname: 'Harbiyeli', lastname: 'Developer', tag: 'Developer', color: '#15996B' },
        { image: 'vezironi.png', firstname: 'Vezir', lastname: 'Oni', tag: 'Staff', color: '#00B2FF'},
        { image: 'harbiyeli.png', firstname: 'Harbiyeli', lastname: 'Developer', tag: 'Trial Staff', color: '#BBFF00' },
    ],
    ```

    ### Music 
    You can add your music to the loading screen by typing the code below.

    In the folder named `config/music` create a folder with the name you put in the path and put a music file in `.mp3` format and the name you put in the path and also put a cover image in `.png` format and the name you put in the path.

    ```js filename="hdstore-loadingscreen/config/config.js" copy showLineNumbers
    ---@param name: Name of the music
    ---@param author: Author of the music
    ---@param path: Path of the music in the `config/music/${path}` folder
    musics: [
        { name: 'CISTAK', author: 'Era7capone ft. Batuflex', path: 'cistak' },
        { name: 'Çözemezler', author: 'Lvbel C5', path: 'cozemezler' },
        { name: 'Kurtlar Sofrası', author: 'Ali Kınık', path: 'kurtlar' },
    ],
    ```
