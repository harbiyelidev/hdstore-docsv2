If you have lost the transaction id, follow these steps.

<Steps>
    ### Go to Tebex
    Go to the <a href={"https://checkout.tebex.io/payment-history/login"} style={{ color: '#008ae6' }}>Tebex</a> page.

    ### Log into Your Account
    Log in with the account you made the purchase with.

    ### Email Verification
    Verify your email address.

    ### Find the Transaction
    Find the transaction you want to get the transaction id from.

    ### Done
    You can now use the transaction id.
</Steps>
